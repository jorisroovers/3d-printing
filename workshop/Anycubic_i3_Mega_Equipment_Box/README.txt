# Summary

I dont have many space around the Printer, so i decide to create a box where i can store my tools and small part. This box fits on the Top Frame of the Printer, so you dont loose much space arround the Printer.

The Clips have two 5mm Pins, they will fit in the 5mm holes of the Box. Maybe its a bit tight, but i got it in without any problems. I heat a knife and pressed the Pins flat like a rivet. So it wont fall of anyway

# Print Settings

Printer: Anycubic I3 Mega
Rafts: No
Supports: Yes
Resolution: 0.3
Infill: 15%

Notes: 
The Supports are only for these 4 Pins on the Clips.
