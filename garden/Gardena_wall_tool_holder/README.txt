# Summary

I was looking for a flexible tool holder for the Gardena combi tool system and did't find anything. So I designed these little gadgets for my tool shed. They will be mounted using two screws (4,5mm or 3/16") in diameter.

I printed mine in ABS at 0,2mm resolution without supports or raft. When you use 30% infill it'll get rock-solid.The print time for my prusa was at approx. 3 hrs per piece.

# Print Settings

Printer Brand: Prusa
Printer: i3 MK2S
Rafts: No
Supports: No
Resolution: 0,2mm
Infill: 30%
